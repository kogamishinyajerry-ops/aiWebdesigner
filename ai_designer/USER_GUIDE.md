# AI Designer - 使用指南

## 🚀 快速开始

### 访问应用

在 Cloud Studio 中：
1. 在右侧面板找到"端口"标签
2. 点击端口 **3000** 的"打开预览"
3. 开始使用 AI Designer

### 首次使用

1. **首页浏览**: 了解所有功能
2. **创建项目**: 点击"开始创作"或"新建项目"
3. **选择工具**: 选择图像、SVG或代码生成器

## 📖 功能使用

### 1. 美学设计引擎 ⭐ 新功能

**路径**: `/generator/aesthetic`

**功能特色**:
- 参考艺术巨匠的作品风格
- 自动生成完整的前端美学方案
- 包含色彩、排版、组件样式、交互效果
- 生成视觉素材的AI提示词

**使用步骤**:
1. 选择艺术风格（梵高、毕加索、达利等11种）
2. 输入页面描述（布局、功能、场景）
3. 选择需要设计的UI组件（Hero Banner、Card、Button等）
4. 可选：设置颜色偏好和情感基调
5. 选择复杂度（低/中/高）
6. 点击"生成美学设计方案"
7. 查看完整的设计方案：
   - **Summary**: 风格分析和色彩预览
   - **Colors**: 完整的色彩方案
   - **Components**: 各组件的CSS和Tailwind代码
   - **Interactions**: 交互动效设计
   - **Assets**: 视觉素材提示词

**支持的艺术风格**:
| 风格 | 艺术家 | 特征 | 适用场景 |
|------|--------|------|----------|
| 梵高 | Vincent van Gogh | 旋涡笔触、浓烈色彩 | 艺术展示、创意产品 |
| 毕加索 | Pablo Picasso | 立体主义、几何碎片 | 科技产品、设计平台 |
| 达利 | Salvador Dalí | 超现实主义、梦境 | 创意工具、沉浸式体验 |
| 莫奈 | Claude Monet | 印象派、光影色彩 | 生活方式、健康应用 |
| 康定斯基 | Wassily Kandinsky | 抽象艺术、几何形状 | 音乐应用、数据可视化 |
| 克利 | Paul Klee | 几何抽象、童趣 | 儿童应用、教育平台 |
| 马蒂斯 | Henri Matisse | 剪纸风格、大胆色块 | 时尚应用、品牌展示 |
| 沃霍尔 | Andy Warhol | 波普艺术、重复图像 | 电商、社交媒体 |
| 埃舍尔 | M.C. Escher | 视错觉、无限循环 | 游戏、创意工具 |
| 歌川广重 | Utagawa Hiroshige | 浮世绘、日式风格 | 文化应用、旅游应用 |

**使用示例**:
```
场景：设计一个瑜伽冥想应用
- 风格选择：莫奈（印象派）
- 页面描述：一个瑜伽冥想应用，追求宁静优雅的氛围
- 组件选择：Hero Banner, Card, Button, Background
- 情感基调：宁静、优雅
- 复杂度：低

输出结果：
- 自然和谐的绿色系色彩方案
- 优雅的渐变背景设计
- 柔和的卡片圆角和阴影
- 流畅的淡入淡出动画
```

### 2. 图像生成器

**路径**: `/generator/image`

**使用步骤**:
1. 在左侧输入提示词，描述您想要的图像
2. 或点击"快捷预设"快速选择模板
3. 选择图像尺寸（1920x1080, 1280x720等）
4. 选择风格（现代简约、科技感、优雅精致等）
5. 点击"生成图像"
6. 等待生成完成，查看预览
7. 点击"下载"保存图像

**提示词示例**:
- "现代科技风格hero banner,渐变背景,抽象几何图形"
- "极简产品展示页面,白色背景,高质感"
- "登录页面背景,磨砂玻璃效果,优雅渐变"

### 2. SVG生成器

**路径**: `/generator/svg`

**使用步骤**:
1. 在左侧输入SVG图形描述
2. 或选择快捷预设（Logo图标、背景图案等）
3. 选择风格（简约、现代、活泼、几何）
4. 选择主色调
5. 点击"生成SVG"
6. 查看SVG预览和代码
7. 下载SVG文件或复制代码

**应用场景**:
- Logo设计
- 图标制作
- 背景图案
- 插画元素

### 3. 代码生成器

**路径**: `/generator/code`

**使用步骤**:
1. 输入UI组件或页面设计描述
2. 或选择预设模板（登录表单、产品卡片等）
3. 选择框架（Tailwind CSS, CSS Modules, Styled Components）
4. 点击"生成代码"
5. 查看生成的代码
6. 复制或下载代码到项目中

**代码类型**:
- React组件
- TypeScript类型
- 可直接使用

### 4. Dashboard

**路径**: `/dashboard`

**功能**:
- 查看统计数据（生成数、使用量等）
- 浏览最近项目
- 快速访问生成器
- 查看使用提示

**标签页**:
- **概览**: 统计和最近项目
- **项目**: 管理所有项目（开发中）
- **分析**: 使用数据分析（开发中）

## 💡 使用技巧

### 获得更好的生成效果

1. **使用详细的描述词**
   ```
   ✅ 好: 现代科技风格hero banner,深蓝色渐变背景,几何图形,玻璃质感按钮
   ❌ 差: 科技背景
   ```

2. **指定尺寸和比例**
   - 宽屏横屏：1920x1080, 1280x720
   - 方形：1080x1080
   - 普通比例：800x600

3. **尝试不同风格**
   - 现代简约：适合商务、科技网站
   - 科技感：适合SaaS、数据分析
   - 优雅精致：适合品牌展示
   - 活泼活泼：适合创意、娱乐

4. **使用快捷预设**
   - 快速开始新项目
   - 了解最佳实践
   - 作为描述词参考

### 工作流建议

1. **批量生成**
   - 使用预设快速生成多个版本
   - 选择最满意的结果

2. **迭代优化**
   - 基于生成结果调整提示词
   - 逐步改进设计

3. **组合使用**
   - 生成图像背景
   - 生成SVG图标
   - 生成代码组件
   - 组合成完整页面

## 🔧 故障排除

### 问题1: 生成失败

**可能原因**:
- 后端服务未运行
- 网络连接问题
- API限制

**解决方法**:
1. 检查后端服务是否运行
2. 查看浏览器控制台错误
3. 刷新页面重试

### 问题2: 生成速度慢

**可能原因**:
- 图像尺寸过大
- 复杂的提示词
- 服务器负载高

**解决方法**:
1. 选择较小的尺寸（1280x720）
2. 简化提示词
3. 等待后处理

### 问题3: 结果不理想

**可能原因**:
- 描述词不够详细
- 风格选择不匹配

**解决方法**:
1. 使用更详细的描述
2. 尝试不同的风格
3. 参考预设模板
