## 🎯 快速开始

### 第一步：启动服务

#### 方式 A：使用启动脚本（推荐）

```
http://localhost:8080/ai_detective/frontend/index.html
```

## 📝 使用流程

### 1. 描述案情

在左侧表单中填写：

#### 案情描述（必填）
详细描述您遇到的问题，包括：
- ✅ 事件发生的时间
- ✅ 事件发生的地点
- ✅ 涉及的人员
- ✅ 具体经过
- ✅ 损失情况

**示例：**
```
未经本人同意，某互联网公司在2023年12月使用我的肖像照片
用于商业广告宣传。照片是在朋友圈发布的个人照片，该公司未经我允许
就下载并使用在他们的产品宣传中。我发现后联系该公司要求删除和道歉，
但对方拒绝。我的肖像权受到了严重侵害。
```

#### 用户信息（选填）
- 姓名
- 联系电话
- 地址

### 2. 开始分析

点击"🔎 开始分析"按钮，AI 将：
- 🔍 提取事实要素
- ⚖️ 识别法律关系
- 📚 匹配适用法律
- 🎯 认定法律责任
- 📊 评估风险

### 3. 查看结果

右侧将显示：
- **事实要素** - 提取的时间、地点、人物等
- **法律关系** - 纠纷类型
- **适用法律** - 相关法律条文
- **法律责任** - 责任主体和依据
- **风险评估** - 成功概率、时间成本、经济成本
- **建议** - 证据收集和诉讼建议

### 4. 生成材料

填写用户信息后，点击"📄 生成材料"按钮，系统将自动生成：
- 📋 **案情说明** - 完整的案件描述
- 📚 **证据清单** - 需要准备的证据列表
- ✍️ **诉求陈述** - 诉讼请求
- 📄 **民事起诉状** - 标准起诉书模板

### 5. 下载材料

点击每个文档右上角的"下载"按钮，保存为 Markdown 文件。

## 📋 支持的案件类型

### 1. 肖像权侵权
**适用场景：** 未经同意使用他人肖像进行商业宣传

**关键词：** 肖像、照片、未经同意、公开、使用、商用

**典型诉求：**
- 停止侵害
- 赔礼道歉
- 赔偿损失

**所需证据：**
- 肖像使用证据
- 肖像权证明
- 损失证明

### 2. 合同违约
**适用场景：** 一方未按照合同约定履行义务

**关键词：** 合同、违约、未履行、不支付、不交付

**典型诉求：**
- 继续履行
- 违约金
- 赔偿损失

**所需证据：**
- 合同原件
- 履约证据
- 违约证据
- 损失证明

### 3. 人身损害
**适用场景：** 造成他人身体、健康损害

**关键词：** 受伤、医疗、住院、误工、残疾

**典型诉求：**
- 医疗费
- 误工费
- 护理费
- 精神损害赔偿

**所需证据：**
- 医疗记录
- 费用票据
- 误工证明
- 伤情鉴定

### 4. 房屋租赁
**适用场景：** 租赁合同纠纷

**关键词：** 租房、房租、押金、违约、退房

**典型诉求：**
- 退还押金
- 支付违约金
- 赔偿损失

**所需证据：**
- 租赁合同
- 付款记录
- 违约证据
- 损失证明

## 💡 使用技巧

### 1. 如何提高分析准确性

✅ **描述详细**
- 提供完整的事件经过
- 包含具体的时间、地点、金额
- 说明各方的行为和态度

✅ **证据提前准备**
- 整理相关合同、协议
- 保留沟通记录（短信、微信、邮件）
- 收集损失证明（发票、收据）

✅ **信息真实**
- 如实描述事件经过
- 不夸大、不隐瞒
- 提供真实联系方式

### 2. 理解分析结果

#### AI 置信度
- **> 80%**: 分析可靠，可参考
- **60%-80%**: 基本可靠，建议补充信息
- **< 60%**: 信息不足，建议详细描述

#### 风险评估
- **成功概率**: 估算胜诉可能性
- **时间成本**: 预计诉讼持续时间
- **经济成本**: 预计费用（律师费、诉讼费等）

### 3. 后续步骤

1. **补充证据**
   - 根据建议收集证据
   - 确保证据的真实性和完整性

2. **咨询律师**
   - 重要案件建议咨询专业律师
   - 获取正式法律意见

3. **准备材料**
   - 填写用户信息
   - 下载生成的材料
   - 补充完善内容

4. **提交申请**
   - 根据材料提交相关部门
   - 跟进案件进展

## 🚨 注意事项

### ⚠️ 免责声明

1. **仅供参考**
   - AI 分析仅供参考，不构成正式法律意见
   - 不替代专业律师的法律服务

2. **重要案件**
   - 涉及大额财产的案件
   - 复杂法律关系的案件
   - 人身损害案件
   建议咨询专业律师

3. **法律时效**
   - 一般民事诉讼时效为 3 年
   - 特殊情况可能不同
   - 注意及时采取法律行动

4. **数据安全**
   - 本地处理，不上传云端
   - 保护个人隐私
   - 及时删除敏感信息

### 🔒 数据安全

- 所有数据处理在本地完成
- 不会上传到第三方服务器
- 建议定期清理浏览记录

## 📞 获取帮助

### 常见问题

**Q: 分析结果准确吗？**
A: AI 基于法律条文和案例模式进行分析，有一定参考价值，但不是正式法律意见。

**Q: 可以直接用生成的材料起诉吗？**
A: 生成的材料是模板，需要根据实际情况修改和完善，重要案件建议律师审核。

**Q: 支持哪些类型的案件？**
A: 目前支持肖像权、合同违约、人身损害、房屋租赁等常见民事纠纷。

**Q: 如何提高分析质量？**
A: 提供详细准确的案情描述，准备充分的证据材料。
