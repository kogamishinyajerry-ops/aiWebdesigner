## ❌ 仍存在的问题

### 1. 事实提取的正则表达式问题

**问题：** 人物提取过多，很多单字被识别为人物

```
人物：王阿
人物：租
人物：房
人物：东
人物：楼
人物：下
人物：邻
人物：居
...
```

**原因：** 正则表达式 `r'([房东|租客|邻居|师傅|楼下邻居])'` 会逐字匹配

**建议：** 使用更精确的正则表达式

### 2. 地点提取不够精确

**当前：**
```
地点：唉，真是倒霉透了，说起来全是泪。大概是去年吧，好像是11月份还是12月份来着，反正就是天冷的时候，我也记不清具体哪天了。那时候我租了个房子，在朝阳区
```

**应该：**
```
地点：朝阳区
```

**原因：** 上下文提取范围太大

### 3. 时间提取不够完整

**当前：** 只提取了"住了一个多月"

**应该：** 还应该提取"去年11月或12月"

**原因：** 时间模式不够全面

### 4. 口语化处理还可以更好

**当前：**
- "水漫金山" → "厨房漏水" ✓
- "说起来全是泪" → "情绪化表达" ✓

**还可以处理：**
- "唉，真是倒霉透了" → 删除（不影响事实）
- "好像"、"可能是" → 标记为不确定信息
- "挺近的" → 可以提取"离地铁近"作为房屋特征
